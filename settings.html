Settings

Note:
1) Focus on the input field you want to change and press the key you want to change it with.
2) Activation time gap is time difference between 2 successive hotkey hits. If the plugin frequently gets activated during your normal typing, consider changing the hotkey or changing the activation gap.


Instructions and Limitations

  • Toggle the highlight mode or the delete mode by double tapping the keys selected in the settings
  • When a particular mode is activated you get a toggle notification
  • In highlight mode select the text you want to highlight
  • In delete mode select as much of the highlighted text as you can to delete that highlight. If you select very less text, the wrong highlight may get deleted
  • The extension will when you try to highlight text spanning across multiple paragraphs or other html elements.
