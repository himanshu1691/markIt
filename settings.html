Instructions and Limitations

  • Extension is disabled on chrome startup so that it doesn't bother you with notifications. Every time you start chrome you have to enable the extension by using the Toggle switch which you can find by clicking on the extension icon on the top right corner. Only then the hotkeys will work.
  • Toggle the highlight mode or the delete mode by double tapping the keys selected in the settings.
  • On the management page, you can export highlights of a particular page and then share the file with other people so that they can import those highlights.
  • The modes have to be activated on every page
  • When a particular mode is activated you get a toggle notification. Try pressing any hotkey twice.
  • In highlight mode select the text you want to highlight
  • In delete mode select as much of the highlighted text as you can to delete that highlight. If you select very less text, the wrong highlight may get deleted.
  • You can see and manage all your highlights on Management page. If you are unable to delete a highlight in delete mode, delete it from the management page.
  • You can turn off the plugin by the global on/off button. You can also filter certain url's from the settings page.
  • The extension will not work when you try to highlight text spanning across multiple paragraphs or other html elements.
  • The extension may not work on certain websites. I will try to improve it over time.
