Version Changelog


The extension will be updated shortly. Here is what has changed:

V 2.0.0
  • Added functionality to export highlights of a page for sharing with others and importing.
V 1.0.5
  • Bugfix for changelog page.
V 1.0.4
  • Bugfix for disabling the extension on startup.
V 1.0.2
  • Changed default setting so that extension is disabled on starup. It has to be enabled from the toggle switch when you want to use it. Then the hotkeys will work.
  • Added changelog page so that users can see the changes made in the new versions.
  • Added function to open settings page when extension is installed for instructions.
V 1.0.1
  • Added function to open extension support page for leaving feedback.
V 1.0.0
  • Added option to turn off extension on all pages
  • Added option to define certain url's on which the plugin won't work
